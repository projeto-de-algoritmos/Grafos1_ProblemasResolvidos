## Instalação 
**Linguagem**: C++<br>

- Compile o código da solução do problema com o comando abaixo:
  
  ```
  $ g++ solution.cpp -o prog
  ```

- Por fim, execute a solução e teste os casos presentes na :

  ```
  $ ./prog
  ```
